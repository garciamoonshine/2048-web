# 🔢 2048 Web

A smooth, polished implementation of the 2048 puzzle game using HTML5, CSS3 and vanilla JavaScript.

## Features
- Classic 4x4 grid gameplay
- Smooth tile slide and merge animations
- Score and best score with localStorage
- Undo last move
- Win screen (reach 2048!) and game over detection
- Swipe support for mobile
- Keyboard arrow key controls

## How to Play
- Use **arrow keys** or **swipe** to slide tiles
- When two tiles with the same number touch, they **merge into one**
- Reach the **2048 tile** to win!

## Controls
| Input | Action |
|-------|--------|
| ← → ↑ ↓ | Slide tiles |
| Swipe | Slide tiles (mobile) |
| Z / Btn | Undo |

## Tech Stack
- HTML5
- CSS3 (transitions, animations)
- Vanilla JavaScript (ES6+)
